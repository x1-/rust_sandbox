# rust_sandbox

Trying to use some crates.

# dependencies

* rust 1.14.0
* rustc-serialize 0.3

# usage

```
$ cargo run
```

# purpose

## fibonacci.rs

original: http://rustbyexample.com/trait/iter.html

Create fibonacci sequences using Iterator.

## rustc_json.rs

Using rustc_serialize::json, to parse str to Json and Struct to str formatted JSON.
